# E-commerce Churn Prediction

Prediction of purchase intent based on user behavioral data from a real-world e-commerce website.

## Dataset
RetailRocket E-commerce Dataset from Kaggle:
https://www.kaggle.com/datasets/retailrocket/ecommerce-dataset

Download the dataset and place the CSV files in the `csv/` folder.

## Project Structure
- `data/` - Data loading, EDA, and preprocessing scripts
- `features/` - Feature engineering
- `models/` - Model training and evaluation

## Pipeline
1. Data Loading & EDA
2. Data Cleaning
3. Feature Engineering
4. Model Training
5. Evaluation
